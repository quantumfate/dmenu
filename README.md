# Installation
```
$ git clone https://github.com/quantumfate/dmenu.git
$ cd into directory
$ sudo make install
```
# Usage
```
$ dmenu_run or dmenu_run_history
```
###### dmenu_run_history will use a script to preselect your recently launched applications
